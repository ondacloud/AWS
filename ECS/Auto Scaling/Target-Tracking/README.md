- ECSServiceAverageCPUUtilization : 평균 CPU 사용량을 정의합니다.  
- ECSServiceAverageMemoryUtilization : 평균 Memory 사용량을 정의합니다.
- ALBRequestCountPerTarget : Application Load Balancer Target Group의 대상별로 완료된 요청을 정의합니다.
